# LBO 模型框架

## 核心假设

### 收购价格
- 通常比当前股价溢价 20-40%
- 控制权溢价反映获得决策权的价值

### 融资结构
- **典型结构**：60% 债务 / 40% 股权
- **债务类型**：
  - 高级担保债务（Senior Secured）：利率 SOFR + 150-250bps
  - 高级无担保债务（Senior Unsecured）：利率 SOFR + 250-400bps
  - 次级债务（Mezzanine）：利率 10-15%

### 债务偿还
- 强制偿还：通常为 EBITDA 的 25-50%
- 自愿偿还：视现金流情况而定
- 目标：5 年内将债务降低 30-60%

### 退出假设
- **退出倍数**：通常假设与进入倍数相同或略低
- **退出年份**：通常 3-7 年
- **退出方式**：出售给战略买家、财务买家、或 IPO

## 输出格式

```markdown
📊 LBO 分析 — [公司名]

收购概况:
• 收购价格: $XX.XX/股（溢价 XX%）
• 企业价值: $X.X billion
• 融资结构: XX% 债务 / XX% 股权

5 年预测:
| 年份 | 营收 | EBITDA | 自由现金流 | 净债务 |
|------|------|--------|-----------|--------|
| Y0 | | | | |
| Y1 | | | | |
| Y2 | | | | |
| Y3 | | | | |
| Y4 | | | | |
| Y5 | | | | |

回报分析:
• 退出企业价值: $X.X billion
• 退出净债务: $X.X billion
• 退出股权价值: $X.X billion
• 股权 IRR: XX%
• MOIC（投入资本倍数）: X.Xx

可行性判断:
• IRR > 20%: ✅ 有吸引力
• IRR 15-20%: ⚠️ 边际
• IRR < 15%: ❌ 不具吸引力
```

## 关键检查点

- [ ] 收购溢价合理（20-40%）
- [ ] 债务/EBITDA 比率在收购时 < 6x
- [ ] 利息覆盖率 > 2.0x
- [ ] 5 年内债务减少 > 30%
- [ ] 退出倍数合理（不高于进入倍数）
- [ ] IRR 计算正确
