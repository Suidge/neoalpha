# 晨会笔记模板

## 格式

```markdown
📋 YYYY-MM-DD 晨会笔记 — 银月
[板块覆盖范围]

Top Call: [标题 — PM 必须听到的一件事]
• 2-3 句核心发展及其意义
• 对股价/目标价的影响

隔夜/盘前动态
• [公司A]: 一句话概括 + 我们的看法
• [公司B]: 一句话概括 + 我们的看法
• [板块/宏观]: 相关板块动态

今日关键事件
• [时间]: [公司] 财报电话会
• [时间]: 经济数据发布（预期 vs 我们的看法）
• [时间]: 会议或投资者日

交易机会（如有）
• [做多/做空] [公司]: 1-2 句论点 + 催化剂
• 风险：什么情况会让这个判断错误
```

## 数据获取

- 市场概况：优先使用 `longbridge-quote` 查询主要指数/ETF。
- 关注标的新闻：优先使用 `longbridge-news`。
- 隔夜新闻：Longbridge 不足时，使用 web_search 搜索 `stock market overnight news today`。
- 宏观日历：使用 web_search 搜索 `economic calendar today`，并标注发布时间。

## 原则
1. 必须有观点 — 只总结新闻不给出看法毫无价值
2. 最重要放最前 — 不要埋没标题
3. "没有新闻"也是有效内容 — 说"隔夜无重大消息，维持现有配置"
4. 区分可操作事件（财报、M&A）和噪音（小型分析师评级变动）
5. 标注时间 — 如果 6am 写的，注明盘前可能变化

6. 涉及交易机会时，末尾加入“仅供研究分析，不构成投资建议或交易指令”。
