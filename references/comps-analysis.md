# 可比公司分析模板

## 数据收集清单

优先读取 [data-layer.md](data-layer.md) 与 `longbridge-peer-comparison`。

1. 确认目标公司和 3–8 个可比公司。
2. 用 Longbridge 获取报价、市值、估值指标、财务报表；复杂命令先查 `--help`。
3. 若 Longbridge 缺少 EV、EBITDA、FCF 等口径，再使用 web_search 搜索可靠来源，并标注来源日期。
4. 不同来源口径冲突时，单独列出，不强行取平均。

## 可比公司选择标准

1. **行业**：同一 GICS 子行业
2. **规模**：市值在同一数量级（±2x）
3. **增长**：收入增长率相近（±10pp）
4. **地域**：主要收入来源地区相同
5. **商业模式**：相似的盈利模式

## 估值倍数说明

| 倍数 | 适用场景 | 注意事项 |
|------|---------|---------|
| EV/Revenue | 亏损或早期公司 | 不考虑利润率差异 |
| EV/EBITDA | 资本密集行业 | 不考虑 CapEx 差异 |
| P/E | 盈利稳定公司 | 不适用于亏损公司 |
| P/FCF | 看重现金流质量 | FCF 波动大的公司慎用 |
| PEG | 成长型公司 | 增长率预测的准确性 |
| EV/GP | SaaS/高毛利公司 | 关注毛利而非营收 |
