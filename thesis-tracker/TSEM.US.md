## 论点陈述

做多 TSEM = 做多 **"光通讯的 picks-and-shovels"** — 不是赌哪条技术路线赢，而是在所有路线都赢。

TSEM 是 AI 光互连产业链里**确定性最高的代工厂**：可插拔、NPO、CPO 三条路线最终都走硅光，都需要硅光代工，而 TSEM 是全球最大的独立硅光代工厂。$13亿的2027年合同、$2.9亿客户预付款、$9.2亿扩产资本支出、2028年更大合同谈判中 — 这些不是预测，是已经签署和正在执行的订单。

**核心投资逻辑链**：
```
AI 算力爆炸 → 光互连需求激增
  → 可插拔 → NPO → CPO 三条路线都用硅光
  → TSEM 是全球最大独立硅光代工厂
  → $13亿合同 + $2.9亿预付款 = 需求已签在纸上
  → 2028年目标：$2.8B营收 / $750M净利
  → 2028E 前瞻 PE 仅 9x = 极度低估
```

## 核心支柱

### 1. 硅光合同锁定增长（确定性最高）📜
- **$13亿硅光合同（已签署）** — 2027年收入的客户合同 + $2.9亿预付款
- **2028年更大合同在谈** — 管理层已确认，2028年客户协议谈判中
- **客户预付款模式** — 客户付钱锁定产能，TSEM 不承担市场风险
- **这是透明度最高的催化剂** — 直接锁定了未来2年增长

### 2. 硅光产能 5倍扩张（先收钱后建厂）🏭
- **$9.2亿资本支出** — 产能建设正在执行
- 到2026年底 SiPho 产能达到2025 Q4的5倍以上
- **70%产能已被客户预留给2028年**
- 这是"先收钱后建厂"的高确定性模式

### 3. Nvidia 战略合作（AI 光互连锁定）🤝
- 与 Nvidia 合作开发 **1.6T数据中心光模块**
- TSEM 的技术被锁定到 AI 最大买家的核心光互连路线图中
- Nvidia 背书 = 技术路线有效性的最强验证

### 4. 2028年目标模型（估值重估空间）🎯
- 管理层目标：$2.8B 营收 + $750M 净利（隐含 ~$6.5 EPS）
- 当前市值仅对应 ~9x 前瞻PE
- 如果兑现，股价有翻倍空间（目标 $450-$500）
- **$920M 资本支出当前正压制利润**，2027-2028年这些产能开始量产时，利润将非线性爆发

### 5. 非技术路线风险暴露（picks-and-shovels）🔧
- TSEM 不赌哪条路线赢
- **可插拔用硅光、NPO用硅光、CPO用硅光** — 三个技术阶段都找 TSEM 代工
- 这是光通讯产业链里最纯的 picks-and-shovels

## 产业链定位

```
SOI（硅光衬底·上游材料）
  ↓
TSEM（硅光代工·核心环节）← 我们在这里
  ↓
LITE / COHR（激光器·光模块组件）
  ↓
GLW（光纤连接·物理基建）
  ↓
NOK（光传输网络·骨干网）
```

TSEM 处于产业链"最不可跳过的一环"——无论下游谁赢，光芯片都要代工。

## 关键风险

1. **技术路线被替代** — 如果 CPO 商业化慢于预期或少用硅光，成长性打折扣。但 TSEM 的可插拔/Datacom 基础业务也能支撑稳定增长。
2. **估值已部分定价** — 从 $37 到 $283（~7.6x），$13亿合同后 5/13-5/14 再涨~15%，短期涨幅已经吸收了部分利好。
3. **资本支出风险** — $9.2亿 capex 如果产能利用率低于预期，ROIC 不达预期。
4. **竞争** — 台积电、GlobalFoundries 等大型代工厂也可能增加硅光产能，减弱 TSEM 先发优势。
5. **以色列地缘风险** — TSEM 总部在以色列，产能涉及全球分布，地缘政治因素不可忽视。
6. **Intel 收购记忆** — 2022年 Intel 收购失败（监管否决）后股价从 $53 暴跌至 $37，市场对此类事件仍有记忆，但当前叙事完全不同。
